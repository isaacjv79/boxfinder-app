Politica de Privacidad

Ultima actualizacion: 30 de diciembre de 2024

Informacion que Recopilamos

BoxFinder recopila la siguiente informacion para proporcionar el servicio:

  • Informacion de cuenta: Email y nombre para crear tu cuenta
  • Contenido del usuario: Fotos de articulos, nombres de contenedores y descripciones que tu proporcionas
  • Datos de uso: Informacion sobre como usas la aplicacion para mejorar el servicio

Como Usamos tu Informacion

  • Para proporcionar y mantener el servicio de BoxFinder
  • Para procesar imagenes con IA y reconocer objetos automaticamente
  • Para sincronizar tus datos entre dispositivos
  • Para permitir funciones colaborativas como equipos compartidos

Almacenamiento de Datos

Tus datos se almacenan de forma segura en servidores protegidos. Las imagenes se procesan para generar etiquetas y miniaturas.

Compartir Datos

No vendemos ni compartimos tu informacion personal con terceros, excepto:

  • Cuando uses la funcion de equipos para compartir contenedores con otros usuarios
  • Cuando sea requerido por ley

Tus Derechos

Puedes solicitar la eliminacion de tu cuenta y todos tus datos en cualquier momento contactandonos.

Permisos de la App

  • Camara: Para tomar fotos de articulos y escanear codigos QR
  • Biblioteca de fotos: Para seleccionar imagenes existentes
  • Internet: Para sincronizar datos y procesar imagenes con IA

Preguntas Frecuentes

¬øComo creo un contenedor?

En la pantalla principal, toca el boton "+" para crear un nuevo contenedor. Dale un nombre y ubicacion.

¬øComo agrego articulos?

Abre un contenedor y toca "Agregar Articulos". Toma una foto y la IA identificara automaticamente el objeto.

¬øFunciona sin internet?

Si, BoxFinder funciona sin conexion. Tus cambios se sincronizaran automaticamente cuando vuelvas a tener internet.

¬øComo comparto contenedores con mi familia?

Ve a Configuracion > Equipos, crea un equipo e invita a otros usuarios por email.
